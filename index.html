# This is a simple HTML file that uses MediaPipe's Hand Landmarker to track hand landmarks from the webcam feed and draw them on a canvas. It loads the model, accesses the camera, and continuously processes video frames to detect hand landmarks, which are then drawn as green circles on the canvas. The status text updates to indicate when the model is ready and when the camera is being accessed. Simple MediaPipe Hand Tracker
